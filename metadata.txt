	There are two buttons in this plugin. 
	1) Select any vector feature from which you want to start navigating or select first feature from attribute table.
	2) Use buttons to navigate to next or prevous features.
	3) Shortcut can also be used to navigate (left key to jump to next feature and right key to jump to previous feature).
